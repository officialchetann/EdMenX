# EdMenX

Education × Management × eXperience

EdMenX is a decision-support system designed to help international students make evidence-based decisions before studying abroad.

The system evaluates:

- Financial sustainability
- Academic compatibility
- Career opportunities
- City affordability
- Transition risks

Germany is used as the initial case study.

Tech Stack:
- Python
- Streamlit
- Pandas
- SQLite
- Plotly
