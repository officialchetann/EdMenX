# EdMenX – International Student Decision Support System

## 📌 Overview
EdMenX is a decision support system designed to help international students make evidence-based decisions before studying abroad.

The system evaluates multiple factors including financial sustainability, academic compatibility, career opportunities, city affordability, and transition risks. Germany is used as the initial case study for implementation and testing.

---

## 🎯 Problem Statement
Choosing the right study destination is difficult due to lack of structured comparison tools and varying socio-economic conditions across cities.

EdMenX solves this problem by providing a data-driven evaluation framework for city selection.

---

## ⚙️ Key Features
- Financial sustainability analysis
- Academic compatibility scoring
- Career opportunity evaluation
- City affordability comparison
- Transition risk assessment model
- Interactive visualization dashboard

---

## 🛠️ Tech Stack
